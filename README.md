## Laravel

```
cd Projet/back-laravel
php artisan serve --host=192........ --port=8000
```

## React
Appli (full dev) sur port 3000: 
```
cd Projet/applifront-react
npm start
```
Appli (build) sur port 5000: 
```
cd Projet/applifront-react
npm run build
serve -s build
```
